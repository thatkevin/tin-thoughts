The Tin
Music and Arts

The Tin logo (not really a logo, but what I did in the summer) has been taken and given a subtle glow effect, referencing the outdoor illuminated sign. The font isn't the same as the actual sign, or brand. It's white on black throughout so it doesn't clash with any of the sub brand colour schemes. I also used the old websites (deliaphonic etc) that I could find old archives of, so I'm guessing a lot, but here is three ways of presenting the 'sub brands'. None of this has been optimised for mobile, which would be the main audience, but I can explain options for that. The example sub sites were generated by AI, so are full of garbage

Design Options

Option 01
The Filing Cabinet
Classic folder tabs. Each brand sits as a raised tab beside the logo. Clean, structured, familiar.
Option 02
The Masthead
Large glowing masthead, editorial feel. Sub brands appear as coloured bands beneath the main header.
THE TIN
Option 03
The Signal
Sleek, modern. The Tin mark stays top-left. Brands switch via pill navigation. Accent line shifts with each brand.
